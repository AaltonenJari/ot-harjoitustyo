# Tiedostoharjoittelua
## Tiedostot
Tässä on harjoiteltu tiedostojen **luontia** ja **muutoksia**. On **lisätty** tekstiä, **poistettu** tekstiä sekä
_lisätty, poistettu ja nimetty tiedostoja uudelleen_. Näin ollaan simuloitu ohjelmistokehitystä.

## git
Tässä yhteydessä on harjoiteltu **Git**-versiohallinnan käyttöä. Tehtiin **paikallinen** repo ja vastaava myös **remoteen**.
Katsotaan miten muutokset näkyvät siellä. Paikalliselle koneelle asennettiin myös **SourceTree**.
